# melonbash
cloudmelon's bash playbook


Recap some BASH commands when playing with Kubernetes and other CNCF projects 

Go home 

    cd ~ 

Check folders and location
   
    ls -l 

Empty the target folder

    rm -rf melonfolder

Unzip a file : 

    tar -xvf ./helm.tar.gz

Move the folder : 

    sudo mv ./helm /usr/local/bin
